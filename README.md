# Match Four in a Row

Real-time multiplayer Connect 4 game.

## How to Play

1. Click **"New Game"** to create a room
2. Share the **6-character code** with your friend
3. Friend clicks **"Join Game"** and enters the code
4. Take turns clicking top of the columns to drop your disc
5. First player to connect **4 discs** wins!
